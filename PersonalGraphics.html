Thesis Project


This is my thesis that I have done on my final year for my Degree it is called the What is the most effective learning experience for students, is it video eLearning or a game-based Learning, or a custom game base with eLearning implementation.This paper goal is to see which learning method is the most effectient when teaching a languague subject. There where three learning method that where tested. These learning methods are game based Learning, custom game based learning and eLearning.



Game Based Learning

This learning method was created with the use of unity game engine, for coding purposes c# was used with the implementatio of realtime database from the services of Firebase. The purpose of this was to keep track of the persons time duration for data collection. The video below is a summary of it of what was used to created. Github Repository



Custom Game Based Learning

This learning method was created with the use of unity game engine, for coding purposes c# was used with the implementatio of realtime database from the services of Firebase. This learning method has the same functionallity of the game based learning one bu the addition of a different information split up and the video learning method is implemented. Accourding to the study this learning method had higher success on on the majority of the group average, median and range. Github Repository
